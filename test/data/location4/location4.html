133 Markie

Contributor Richard Bannister

Grade 3+

Length 2km

Start NN 577 953 (57.0265, -4.3459)

Finish NN 581 938 (57.0133, -4.3383)

A decent introduction to walk-in ‘creek boating’ for intermediates, this run has some lively water in a low gorge, but is never too serious. You’ll need a good bit of water: other rivers like the Roy will be running. Walk up the track, river left, from below the Spey Dam. Put on wherever you fancy. Don’t paddle it thinking you are on the Feith Talagain, like we did, constantly thinking there will be ‘gnar’ just around the next corner. There isn’t.
